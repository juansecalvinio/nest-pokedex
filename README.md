### Ejecutar en desarrollo

1. Clonar el proyecto
2. Ejecutar `yarn install` o `npm install` para instalar las dependencias
3. Instalar `@nestjs/cli` con:

```
npm i -g @nestjs/cli
```

4. Levantar la base de datos:

```
docker-compose up -d
```

### Stack utilizado

- MongoDB
- NestJS
